# Docker Image for Spark History Server

Docker image for Spark history Server on Kubernetes

## Supported tags

`aws-v3.0.1` `aws-v2.4.0` `aws-latest` 

`gcp-v3.0.1` `gcp-v2.4.0` `gcp-latest` 

`azure-v3.0.1` `azure-v2.4.0` `azure-latest` 

## Pull image

### Github Docker

